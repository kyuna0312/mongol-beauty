# Rem & Ram Anime Girl Images

## How to Add Images

To use Rem and Ram anime girl images in error displays, add these files to the `public` folder:

1. **`rem-anime.png`** - Rem character image (recommended: 256x256px or larger, circular crop)
2. **`ram-anime.png`** - Ram character image (recommended: 256x256px or larger, circular crop)

## Image Requirements

- Format: PNG (with transparency preferred)
- Size: 256x256px or larger
- Style: Cute anime girl (Rem & Ram from Re:Zero)
- Background: Transparent or solid color

## Fallback

If images are not found, the system will automatically show emoji fallbacks:
- 💙 for Rem
- 💜 for Ram

## Where Images Are Used

- ErrorBoundary (full page errors)
- ErrorDisplay component (major errors)
- InlineError component (form errors)

You can find free Rem & Ram images from:
- Official Re:Zero artwork
- Fan art (with proper attribution)
- Anime character image databases
