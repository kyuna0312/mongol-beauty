# Security Policy

## Reporting a Vulnerability

To report a security vulnerability, email **khatanzorigb@gmail.com** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

You can expect an acknowledgement within 48 hours. Please do not open a public GitHub issue for security vulnerabilities.

## Scope

This policy covers the `mongol-beauty` monorepo, including the NestJS API, React frontend, and shared packages.

## Out of Scope

- Vulnerabilities in third-party dependencies (report those upstream)
- Issues already known and tracked in GitHub Issues
